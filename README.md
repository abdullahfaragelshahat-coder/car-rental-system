# 🚗 نظام إدارة تأجير السيارات (Car Rental System)

نظام إلكتروني متكامل لإدارة عمليات تأجير السيارات، تسجيل العملاء، ومتابعة الحجوزات.

## 🚀 المميزات
* **إدارة السيارات:** إضافة وتعديل حالة السيارات (متاحة/مؤجرة).
* **إدارة العملاء:** تسجيل بيانات العملاء ونظام نقاط الولاء (VIP).
* **نظام الحجوزات:** تسجيل حجوزات جديدة مع تحديث حالة السيارة أوتوماتيكياً.
* **إشعارات وتقارير:** متابعة يومية للعمليات وتقارير الإيرادات.
* **لوحة تحكم:** واجهة بسيطة وسهلة للموظفين.

## 🛠 التقنيات المستخدمة
* **اللغات:** PHP, HTML5, CSS3, JavaScript.
* **قاعدة البيانات:** MySQL.

## ⚙️ كيفية التشغيل
1. قم بإنشاء قاعدة بيانات جديدة وقم باستيراد ملف الـ SQL الخاص بالمشروع.
2. عدل بيانات الاتصال بقاعدة البيانات في ملف `db.php`.
3. قم بتشغيل المشروع على خادم محلي (مثل XAMPP).
4. ادخل إلى لوحة التحكم عبر ملف `admin.php`.

## 🛡 الأمان
* تم تصميم النظام بآليات لمنع الأخطاء وتحديث البيانات بشكل آمن لضمان دقة التقارير.
